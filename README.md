# HarvestMind 🌾

Decision Intelligence for Farmers — powered by GLM (z.ai)

## How to Run

### Step 1 — Make sure Node.js is installed
Download from https://nodejs.org if you don't have it.

### Step 2 — Open this folder in VS Code terminal
Press Ctrl + ` to open the terminal, then:

```
cd path/to/harvestmind
```

### Step 3 — Start the server
```
node server.js
```

You should see:
```
🌾 HarvestMind running at http://localhost:3000
```

### Step 4 — Open in browser
Go to: http://localhost:3000

That's it! The chat will now work properly.

## Files
- server.js        → Node.js backend (handles API calls to z.ai)
- public/index.html → Frontend (the chat UI)
- package.json     → Project info
